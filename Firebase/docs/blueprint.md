# **App Name**: CoursePilot

## Core Features:

- Course Chatbot: A chatbot grounded in course materials that can answer student questions, suggest next topics, and provide personalized quizzes, acting as a virtual teaching assistant.
- Learning Trajectory Definition: Teachers can define a structured learning trajectory by creating a list/graph of topics and prerequisites.
- Resource Tagging: Teachers can upload course resources and tag them to specific topics, linking materials with topics and making them available to the chatbot and practice quiz generator.
- Progress Monitoring: Teachers can track student progress along the learning trajectory by tracking quiz scores.
- Personalized Practice: Students can get personalized practice quizzes and exercises for a specific topic.
- Motivational Explanations: LLM tool to give students explanations of why topics matter and how they connect to real-world problems, enhancing motivation and engagement.
- Topic Recommendations: System to recommend what to learn next, grounded on the learning trajectory.

## Style Guidelines:

- Primary color: Strong, trustworthy blue (#3B82F6) for knowledge and stability.
- Background color: Light, desaturated blue (#E0E7FF) for a clean, calming interface.
- Accent color: Energetic purple (#8B5CF6) to highlight important information.
- Headline font: 'Space Grotesk', a geometric sans-serif font, will give the site a futuristic, yet professional appearance. Body text: 'Inter' will ensure the site is readable on a variety of screen sizes; sans-serif.
- Code font: 'Source Code Pro' monospace font for code snippets in a clean and readable manner.
- Icons should be modern and simple, related to learning, progress, and course management.
- A clear and intuitive layout with distinct sections for the learning trajectory, chatbot, and teacher dashboard.

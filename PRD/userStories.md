# User Stories — Team LLMRnD

## Persona: Student
- As a student, I want to ask course-related questions and receive Socratic hints instead of direct answers.  
- As a student, I want to practice topics with 3–5 quick questions and immediate feedback.  
- As a student, I want to see how well I understand each topic (Needs Practice / Getting There / Strong).  

## Persona: Teacher
- As a teacher, I want to upload course materials (slides, notes, assignments).  
- As a teacher, I want to generate practice questions for each topic automatically.  
- As a teacher, I want to see anonymized summaries of student performance.  
- As a teacher, I want the system to reject questions that are outside course scope.

# User Stories:

**Socratic Chatbot Interface**
* **User Stories:**
    * As a student, I want to ask questions about course material and receive guided hints so that I learn to solve problems independently.
    * As a student, I want the system to detect my misconceptions so that I can correct my understanding.
    * As a student, I want the chatbot to stay focused on course material so that I don't waste time on unrelated topics.

**Pre/Post Assessment System**
* **User Stories:**
    * As a student, I want to know my starting knowledge level on a topic so that I understand what I need to focus on.
    * As a student, I want to measure my improvement after learning so that I feel confident in my progress.
    * As the system, I need baseline and post-interaction knowledge data to measure learning effectiveness.

**Course Material RAG System**
* **User Stories:**
    * As a teacher, I want to upload course materials so the chatbot can reference official content.
    * As a student, I want chatbot responses based on course materials.
    * As the system, I need to retrieve relevant course content to ensure responses are scoped correctly.

**Basic Progress Tracking**
* **User Stories:**
    * As the system, I need to track student interactions to measure learning effectiveness.
    * As a student, I want to see my progress history.
    * As a teacher, I want anonymized aggregate data to understand class performance.

**Adaptive Difficulty System**
* **User Stories:**
    * As a student, I want more direct hints when I'm stuck.
    * As a student, I want the system to challenge me more when I'm doing well.
    * As the system, I need to adapt to individual student ability.

**Anonymized Teacher Dashboard**
* **User Stories:**
    * As a teacher, I want to see which topics students struggle with.
    * As a teacher, I want to identify common misconceptions.
    * As a student, I want my privacy protected.

**Auto-Generated Quiz Bank**
* **User Stories:**
    * As a teacher, I want the system to generate quiz questions to save time.
    * As a teacher, I want to review and edit generated questions.

**LLM-Resilient Assignment Generator**
* **User Stories:**
    * As a teacher, I want to create assignment variations so students cannot share solutions.
    * As a teacher, I want variations to be truly different.

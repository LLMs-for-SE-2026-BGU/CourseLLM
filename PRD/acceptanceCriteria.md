# Acceptance Criteria — CourseLLM MVP

## Functional
- [ ] Chatbot restricts answers to course materials only  
- [ ] Each answer includes at least one source citation  
- [ ] Teachers can upload ≥3 materials successfully  
- [ ] Students can complete a 5-question practice session  

## Technical
- [ ] Response time ≤ 6s (avg)  
- [ ] System runs fully via Docker Compose  
- [ ] Database integrity checks pass  

## Educational
- [ ] ≥ 20% average improvement in topic practice scores  
- [ ] ≥ 50% weekly active users in pilot course  
- [ ] Teachers identify ≥ 3 weak topics via dashboard
